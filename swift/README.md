# Swift Explorations

To get Swift:

_Swift comes with XCode_

To run the tests on a Unix-like shell:

```
./test.sh
```

You can run many of the individual examples in one shot:

```
swift triple.swift
```

or compile and run separately:

```
swiftc triple.swift && ./triple
```
